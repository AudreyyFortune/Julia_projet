## How to use the script 'projet_long_Julia.ipynb'
  1. Installation
  
 ```
julia> using Pkg
julia> Pkg.add("IJulia")
```
  2. Usage
  
In a Julia shell :
```
julia> using IJulia
julia> notebook()
```


## How to use the script 'projet_long_Python.ipynb'
  1. Installation
  
With conda :
```  
conda install -c conda-forge notebook
```
With pip :
```
pip install notebook
```

  2. Usage
  
In a Python shell :
```
jupyter notebook
```
